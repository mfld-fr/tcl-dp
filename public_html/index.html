Tcl-DP Distributed Programming for Tcl

After nearly a decade without updates, Tcl-DP is back. Tcl-DP has been updated to work with the latest versions of Tcl as well as the most recent operating systems, while maintaining compatability with Tcl-DP 4.0.

Tcl-DP adds TCP, UDP, IP-multicast connection management, remote procedure call (RPC), serial communication and distributed object protocols to the Tcl language.

A C API allows non-Tcl RPC clients to send commands to an RPC server and receive responses.

See each version's Release Notes for the exciting details.
