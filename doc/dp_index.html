Tcl-DP v3.4 Manual

An overview of Tcl-DP and dpwish.

Tcl-DP commands:

  • dp_address - Manipulating Internet addresses
  • dp_atclose - Running commands upon close of file or socket
  • dp_atexit - Running commands upon termination of program script
  • dp_connect - Opening, closing, reading and writing to/from sockets
  • dp_distribObj - Distributed object interface
  • dp_filehandler - Handling file descriptor conditions
  • dp_object - Support for procedural objects
  • dp_rpc - Remote Procedure Call support
  • dp_socketOption - Socket configuration and information
  • traceProc - Procedure tracing
