Tdp_PacketReceive, Tdp_PacketSend - C level access to "dp_packetSend" and "dp_packetReceive" commands

#include "network.h"

int Tdp_PacketReceive (interp, fileHandle, peek)

int Tdp_PacketSend (interp, fileHandle, message)
Tcl_Interp   *interp         /* Current Tcl interpreter */
char         *fileHandle     /* Socket to send or receive data from */ 
int          peek            /* Destructive read? */
char         *message        /* String to send */

The above two routines allow users C level access to the Tcl-DP dp_packetSend and dp_packetReceive Tcl/Tk commands.
